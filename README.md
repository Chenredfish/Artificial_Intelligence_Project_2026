# Artificial Intelligence Project 2026

A board game program for a 2-player 8x8 strategy game. See [RULE.md](RULE.md) for full game rules.

## Goal

Build a **legally compliant move system** — no AI strategy required at this stage.

## Milestones

- [ ] Board representation (8x8 grid, piece placement)
- [ ] Move generation (validate legal moves per piece type)
- [ ] Input handling (keyboard / mouse / text file)
- [ ] Output: coordinate notation + board state diagram
- [ ] Turn management (AB first, UV second, alternating)
- [ ] Capture logic (land on opponent's piece to capture)
- [ ] Timer tracking (60s limit, 3 extensions to 120s)
- [ ] Game end detection (20 rounds or disqualification)
- [ ] Score calculation (A/B/U/V=3pts, others=1pt)
